Exantara

Exantara, or Exploration of the Archipelago, is a website that provides information about Indonesia's rich cultural heritage, including traditional clothing, signature dishes, and regional dances. Built using PHP and Tailwind CSS, the website features two interfaces: the client side, which allows users to access a variety of cultural information, and the admin side, which facilitates the addition of new data to update content regularly.

UI/UX Project

This project showcases the UI/UX design for Dubu Air, a flight booking application equipped with additional features such as real-time weather forecasts for travel destinations and passenger reviews to provide feedback from their experiences. With a focus on ease of use and aesthetics, this design aims to create a seamless and informative experience for users in planning their trips.

Security Planners

Security Planners is a Java Swing-based application developed in NetBeans to optimize organizational security management. It features an admin dashboard, personnel management (add, update, delete staff details), scheduling (shifts, dates, locations), and location assignments (e.g., main gates, back entrances). The app leverages OOP principles with structured classes like Admin, Security Personnel, Location, and Schedule, ensuring efficient and user-friendly security operations.

Community Health Center Service System

This project implements a healthcare service system using C++ that facilitates patient queue management, data recording, prescription tracking, and efficient disease categorization. This project aims to improve the efficiency and quality of healthcare services, particularly in managing patient queues and systematically documenting medical data.
